# 용스튜 트친소/소개표 생성기
![Version](https://img.shields.io/badge/version-v0.5.0-blue)
![Status](https://img.shields.io/badge/status-prototype-orange)

트친소표를 보다 편리하게 제작하고 공유하기 위해 만들어진 웹 기반 생성기입니다.

🔗 트친소/소개표 생성기 사용하기
https://rgg-friend.vercel.app/

---

## 프로젝트 소개

용스튜 트친소표 및 소개표를 웹에서 간편하게 제작할 수 있도록 만들었습니다.

별도의 이미지 편집 프로그램 없이 원하는 정보를 입력하고 트친소 및 소개표를 생성할 수 있습니다.

업데이트는 제작자의 일정이 될 때 진행됩니다.

이용 문의 및 건의사항은 #용스튜소개표_제보 또는 하기 메일로 알려주세요.

불호 항목의 경우 현재 이상 추가 계획 없음(건의 안받음)
